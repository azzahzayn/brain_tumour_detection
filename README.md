# brain_tumour_detection
This project is a deep learning-based system for detecting brain tumors from MRI images done using Convolutional Neural Networks (CNN) and transfer learning done using MobileNetV2. The system classifies MRI scans into two categories:  
- Tumour  
- No Tumour  

**The project includes:**  
- Model comparison  
- Accuracy visualization  
- Confusion matrix  
- Classification report  
- GUI dashboard for prediction  

**Features**
- Brain tumor classification using CNN  
- Transfer learning using MobileNetV2  
- MRI image preprocessing  
- Accuracy and validation graphs  
- Confusion matrix visualization  
- Precision, Recall and F1-score evaluation  
- Multi-image prediction dashboard  
- Prediction confidence display  

**Dataset link** : [Clickable Text] https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection  

**CNN Architecture:**  
- Convolution layers  
- MaxPooling layers  
- Flatten layer  
- Dense layer  
- Dropout layer  
- Sigmoid output layer

**Future Improvements:**  
- Larger MRI datasets  
- Web application deployment  
- Multi-class tumor classification
